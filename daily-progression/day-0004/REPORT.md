# Day 0003 — Time Series Analysis & Financial Metrics

## What I Learned

### Terms / Concepts:
- Correlation — measure of how two assets move together (-1 to +1)
- Correlation matrix — all pairwise correlations in a table
- Portfolio weights — fractions of capital allocated to each asset
- Portfolio return — weighted average of individual asset returns
- Dot product — mathematical operation multiplying arrays element-by-element then summing
- Portfolio return - The blended daily return of your whole portfolio, weighted by how much you hold of each stock.
